# IIoTDataCollector-
CourseWork 
## Стек технологий
* Backend демонстрационной системы: Python, Flask  
* Frontend демонстрационной системы: HTML,CSS,JS
* Сервер модуля сбора данных: Python
    + pymodbus 
    + influxdb-client 
* M2M протокол: Modbus TCP
* РСУБД: PostgeSQL
* Time series database: InfluxDB
